  • Note blocks to command blocks (1.13.+)
    With this plugin for MCedit you can replace the music blocks into the command blocks. To have the player hear the music you need: 1) Create a scoreboard called music ("/scoreboard objectives add music dummy"). 2) Set the score for the player who will hear this music ("/scoreboard player set music 1"). All is ready!
    Published: When version 1.13 is released
